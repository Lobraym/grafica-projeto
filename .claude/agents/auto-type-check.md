# Auto Type Check

1. Execute `npx next build` para verificar erros de tipagem.
2. Busque por usos de `any` nos arquivos modificados recentemente.
3. Reporte erros encontrados com arquivo e linha.
4. Se não houver erros, confirme que está tudo ok.
